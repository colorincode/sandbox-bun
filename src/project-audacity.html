The Audacity Project

Audacity was a project born out of imagination. Color in Code handled the product design from conceptualization to manufacture, the graphic design and digital marketing materials, as well as the main website, which was set up through Wordpress ™ and WooCommerce ™. This website concept is the periodic table of elements as the foreground and the LA city skyline as the background.
